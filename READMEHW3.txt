How to run program:
- on just place all of the files in the same folder, cd to the folder in cygwin, and then type
	g++ hw3.cpp
	./a.exe

- To test the prefix to binary tree function you just need to change the string passed into the function prefixToTree(). When you enter an equation into this string, every symbol, number, or letter must be followed by a space except for the last number.

- As for the count and insert functions you can just make calls to these with your own parameters on a bag object.

- To test the erase function just call it on a bag object.
